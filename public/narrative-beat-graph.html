Beats & Edges

Each Beat (node) is a scene, moment, or narrative unit. Edges are the choices or transitions between them.

Tags

Tone, location, and faction tags let you see the narrative shape at a glance. Tone sets the node colour on the canvas.

Requirements

A requirement must be true for a path to validly reach this node. Flag requirements check a boolean key: key = true/false. Stat requirements compare a number: stat ≥ 5.

Effects

Effects fire when the beat is reached. Flags are set to a value. Stat effects add a delta (use negative numbers to subtract). Reputation effects change a faction's standing.

Propagation

The engine does a DFS from all start nodes (nodes with no incoming edges), accumulating state along every path. It then checks each node's requirements against every possible state at that node. Any requirement that no path satisfies is flagged as a contradiction.

Controls

Double-click canvas → new beat. Shift-drag from node → draw edge. Right-click → context menu. Scroll → zoom. Middle-drag / drag background → pan.
